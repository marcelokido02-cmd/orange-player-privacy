Política de Privacidade - Orange Player

Última atualização: 29 de março de 2026

Bem-vindo ao Orange Player. Esta Política de Privacidade explica como lidamos com suas informações, garantindo transparência total sobre o uso de dados para anúncios e análise de desempenho conforme as normas do Google Play.

1. Acesso a Arquivos e Permissões

O Orange Player solicita permissões estritamente necessárias para sua função de reprodutor de mídia local:

  • Arquivos de mídia: Para localizar e reproduzir seus vídeos e áudios.
  • Gerenciamento de Pastas (SAF): Para que você possa organizar, renomear ou excluir seus arquivos sob comando direto.
  • Rede e Internet: Necessário para carregar anúncios e enviar dados de diagnóstico anônimos.

2. Publicidade (Google AdMob)

Este aplicativo exibe anúncios através do Google AdMob. Para fornecer publicidade relevante, o serviço pode coletar e utilizar:

  • ID de Publicidade do Android: Um identificador anônimo usado para personalizar anúncios e evitar repetições.
  • Dados de Localização Aproximada (IP): Usado para exibir anúncios no idioma correto e relevantes para sua região (ex: ofertas no Brasil).

3. Análise e Desempenho (Google Firebase)

Utilizamos o Firebase Analytics para monitorar a estabilidade do app. Coletamos dados técnicos anônimos como:

  • Modelo do aparelho (ex: Lenovo IdeaPad) e versão do Android.
  • Interações com a interface (ex: cliques no botão de pausa).
  • Relatórios de falhas (Crashlytics) para corrigir bugs rapidamente.

4. Compartilhamento com Terceiros

Os dados técnicos e de publicidade são processados exclusivamente pelo Google. Não vendemos nem compartilhamos suas informações com corretores de dados ou outras redes de terceiros. Você pode gerenciar suas preferências de anúncios nas configurações do seu dispositivo Android.

5. Segurança e Retenção

Todos os dados de diagnóstico são criptografados em trânsito via HTTPS. Como não coletamos nomes ou e-mails, não armazenamos dados de identidade pessoal em nossos servidores.
